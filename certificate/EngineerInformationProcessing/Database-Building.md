# 데이터베이스 구축



### SQL의 분류

- DDL(데이터 정의어)
  - CREATE
  - ALTER
  - DROP
  - TRUNCATE
- DML(데이터 조작어)
  - SELECT
  - INSERT
  - UPDATE
  - DELETE
- DCL(데이터 제어어)
  - GRANT
  - REVOKE

- TCL(트랜잭션 제어어)
  - COMMIT
  - ROLLBACK
  - CHECKPOINT

### 제약조건 유형

- PRIMARY KEY(기본키)
- FOREIGN KEY(외래키)
- UNIQUE KEY(유일키)
